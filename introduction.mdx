## What is CCReward?

CCReward is a comprehensive credit card rewards optimization tool designed for users in **India** and **Singapore**. It solves the common problem of figuring out which credit card to swipe for maximum rewards on any given purchase.

The app supports over 40 banks across both countries and hundreds of individual credit cards, with real-time reward calculations based on Merchant Category Codes (MCCs).

## Mobile apps

CCReward is available on iOS and Android in addition to the web app at [ccreward.app](https://ccreward.app).

- **iOS**: Available on the [App Store](https://apps.apple.com/in/app/ccreward/id6736835206)

## Self-hosting

CCReward is open source for non-commercial use. If you want to run your own instance, see the [installation guide](/setup/installation) for setup instructions including Firebase configuration and environment variables.

<Note>
  The live app at [ccreward.app](https://ccreward.app) is always up to date with the latest bank and card data. Self-hosted instances require manually fetching static data via `npm run fetch-data`.
</Note>
